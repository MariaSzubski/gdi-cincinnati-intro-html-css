WHAT IS CSS?

CSS: What is it?

CSS = Cascading Style Sheets

CSS is a "style sheet language" that lets you style the elements on your page.

CSS is works in conjunction with HTML, but is not HTML itself.

CSS: What can it do?

All colors, text, position, and size

CSS: What does it look like?

The CSS Rule

The CSS Rule

selector {
  property: value;
  property: value;
}
  • A block of CSS code is a rule.
  • The rule starts with a selector.
  • It has sets of properties and values.
  • A property-value pair is a declaration.

CSS Syntax

Selectors tie our CSS rules to specific HTML elements.

selector {

}

"I want to select all the paragraphs."

"I want to select the list of recipe ingredients."

CSS Syntax

Declarations are property-value pairs that define the styles applied to an HTML element. 

selector {
  property: value;
  property: value;
  property: value;
}
  • Properties and values are separated by a colon :
  • Each declaration ends with a semicolon ;
  • Declaration blocks are surrounded by curly brackets { }

Types of Selectors

Element

Use the HTML tag name without angle brackets < > 

p {
  property: value;
}
Selects all paragraph elements. 
img {
  property: value;
}
Selects all image elements.

Connecting CSS to HTML

3 Options:

"Inline"

"Embedded"

"External"

Connecting CSS to HTML

Inline

<p style="color:red">Some text.</p>
  • Use the style="" attribute to add CSS to an HTML element
  • CSS declarations are placed inside the quotations " "
  • Difficult to use and update in large projects
  • Not preferred

Connecting CSS to HTML

Embedded

<head>
  <style type="text/css">
    p {
      color: blue;
      font-size: 12px;
    }
  </style>
</head>
  • CSS is placed inside the <style> tag
  • Nested inside <head> element in an HTML file
  • Uses standard CSS syntax
  • Can only be used by one html file

Connecting CSS to HTML

External

<head>
  <link href="styles.css" rel="stylesheet" type="text/css">
</head>
  • Create a separate .css file for CSS rules
  • Link to the external stylesheet in the <head> tag
  • Shared resource for multiple pages
  • Reduces ﬁle size & bandwidth
  • Easy to maintain in larger projects
  • Preferred by nerds everywhere!

CSS Color Values

Your browser can accept colors in many different ways:

  • Color name: red
  • Hexadecimal value: #FF0000
  • RGB value: rgb(255, 0, 0)
  • HSL value: hsl(0, 100%, 100%)

The 16 standard color keywords are:
black, gray, maroon, purple, green, olive, navy, blue,
silver, white, red, fuchsia, lime, yellow, teal, and aqua.

CSS Properties

Property: Color

The color property changes the color of text.

p {
  color: red;
  color: #ff0000;
  color: rgb(255, 0, 0);
}

Property: Background-color

The background-color property changes the color of the background.

p {
  background-color: black;
  background-color: #000000;
  background-color: rgb(0,0,0);
}

Multiple Property Values

Some properties like background and font-family can have one or more comma separated values.

p {
  color: white;
  font-family: Arial, sans-serif;
  background-image: url('img/bubbles.png'), url('img/stars.png');
}

Property: Font-family

The font-family property deﬁnes which font is used.

p {
  font-family: "Trebuchet MS";
  font-family: serif;
  font-family: "Arial", sans-serif;
}
  • Can be a Generic or Specific font name
  • Define alternative fonts in a comma-separated list
  • Wrap font names with spaces in matching
    quotations " " or ' '.

Reusing code

As a general coding principle, Don't Repeat Yourself.

To reuse CSS, we use IDs and classes.

Types of Selectors

IDs vs. Classes

ID

  • Should be unique to one element on a webpage
  • Begins with # 
    #footer { }

Class

  • Lots of elements can have the same class
  • Begins with . 
    .warning { }

Types of Selectors

ID

#footer {
  background-color:#1A6687;
}
Selects the element with an id of "footer". 
<p id="footer">© Copyright 2011</p>
The associated HTML uses the id="" attribute.

© Copyright 2011

Types of Selectors

Class

.warning {
  color: red;
}
Selects all elements with a class of "warning". 
<p class="warning">Run away!</p>
The associated HTML uses the class="" attribute.

Run away!

Cascading priority

Your browser assigns different priorities to CSS depending on the type of selector.

  1. !Important - Most Important
  2. Inline CSS
  3. ID
  4. Class
  5. Element - Least Important

Cascading priority

Your browser also assigns priority based on the specificity of the selection. More specific selectors have higher priority.

.main .sale .clearance p { /* Most specific */
  color: red;
}
.header .title p {
  color: green;
}
.footer p { /* Least specific */
  color: blue;
}

Cascading priority

The tie-breaker is position. Rules lower in the file overwrite rules higher in the file

a {
  background-color: yellow;
}

a {
  background-color: teal;
}

a { /* This rule wins */
  background-color: black;
}
